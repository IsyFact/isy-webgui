Dies ist eine Beispielseite für modale Dialoge. Durch Klicken auf den folgenden Button wird ein Dialog geöffnet. Sollte kein Java-Script aktiviert sein, so wird der Dialog direkt im Fenster angezeigt.




Durch Klicken auf den nächsten Button wird ein beispielhafter Bestätigungsdialog geöffnet. Bestätigungsdialoge sind spezielle modale Dialoge, die als Subflows eingebunden werden. Die anzuzeigenden Texte und Knopfbeschriftungen werden dem Flow als Parameter übergeben. Bestätigungsdialoge sind daher einfach und vielfältig einsetzbar und eignen sich immer, wenn der Benutzer vor eine "Ja/Nein Entscheidung" gestellt werden soll. Sollte kein Java-Script aktiviert sein, so wird der Dialog direkt im Fenster angezeigt.
