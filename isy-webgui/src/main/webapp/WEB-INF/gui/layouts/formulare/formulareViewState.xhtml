Formulare

Formularinhalte werden mit dem Tag <div clas="form-horizontal"> eingeleitet. Zeilen über <div class="row"> . Einzelne Spalten über das Bootstrap Grid Layout (z.B. für drei Spalten, drei Elemente <div class="col-lg-4"> )

Für einzelne Formularelemente stehen die <isy:form... Komponenten zur Verfügung. Diese enthalten neben dem eigentlichen Feld ein Label und integrieren sich in das Form-Layout.

 Readonly-Formulare

Formular können ohne größeren Aufwand in einen readonly-Modus versetzt werden. Dafür muss der Readonly-Modus der entsprechenden <isy:form... Elemente aktiviert werden. Weiterhin muss dem übergeordneten form-horizontal Element noch die CSS-Klasse readonly zugewiesen werden.
